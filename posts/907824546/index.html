жң¬ж–Үд»Ӣз»ҚдәҶеҰӮдҪ•йҖҡиҝҮ Mono е®һзҺ° .NET зЁӢеәҸзҡ„и·Ёе№іеҸ°иҝҗиЎҢпјҢе№¶жҺўи®ЁдәҶжҗӯе»әиҪ»йҮҸзә§зҡ„ .NET иҝҗиЎҢж—¶зҡ„еҸҜиғҪжҖ§гҖӮйҰ–е…ҲпјҢж–Үз« и®Ёи®әдәҶ .NET зүҲжң¬е…је®№й—®йўҳпјҢжҸҗеҮәдәҶйҖҡиҝҮеә”з”ЁзЁӢеәҸй…ҚзҪ®ж–Үд»¶дёӯзҡ„supportedRuntime иҠӮзӮ№жқҘжҢҮе®ҡеә”з”ЁзЁӢеәҸе…је®№зҡ„ .NET Framework зүҲжң¬гҖӮйҡҸеҗҺпјҢж–Үз« д»Ӣз»ҚдәҶ Mono иҝҗиЎҢж—¶зҡ„жҗӯе»әпјҢеҢ…жӢ¬еҝ…иҰҒзҡ„з»„д»¶е’Ңзӣ®еҪ•з»“жһ„пјҢд»ҘеҸҠеҰӮдҪ•дҪҝз”Ёе‘Ҫд»ӨиЎҢжҲ– Mono Embedding ж–№жЎҲжқҘиҝҗиЎҢ .NET зЁӢеәҸгҖӮж–ҮдёӯиҝҳжҸҗдҫӣдәҶдёҖдёӘдҪҝз”Ё C++ жЁЎжӢҹе‘Ҫд»ӨиЎҢжү§иЎҢ Mono иҝҗиЎҢж—¶зҡ„зӨәдҫӢд»Јз ҒпјҢе№¶и®Ёи®әдәҶеҰӮдҪ•йҒҝе…Қе‘Ҫд»ӨиЎҢзӘ—еҸЈеј№еҮәзҡ„й—®йўҳгҖӮжңҖеҗҺпјҢж–Үз« жҖ»з»“дәҶдҪҝз”Ё Mono е®һзҺ°иҪ»йҮҸзә§ .NET иҝҗиЎҢзҺҜеўғзҡ„дјҳеҠҝе’ҢеӯҳеңЁзҡ„й—®йўҳпјҢд»ҘеҸҠеҸҜиғҪзҡ„и§ЈеҶіж–№жЎҲгҖӮ

еңЁдҪҝз”Ё Mono и®©.NET зЁӢеәҸи·Ёе№іеҸ°иҝҗиЎҢиҝҷзҜҮж–Үз« дёӯпјҢжҲ‘д»¬е·Із»ҸеҜ№ Mono д»ҘеҸҠ.NET зЁӢеәҸзҡ„иҝҗиЎҢжңәеҲ¶жңүдәҶеҲқжӯҘзҡ„зҗҶи§ЈгҖӮд»ҠеӨ©жҲ‘жғіжқҘи°Ҳи°Ҳ"дҪҝз”Ё Mono жү“йҖ иҪ»йҮҸзә§зҡ„.NET иҝҗиЎҢж—¶"иҝҷж ·дёҖдёӘиҜқйўҳгҖӮдёәд»Җд№ҲжҲ‘дјҡжңүиҝҷж ·дёҖз§Қжғіжі•е‘ўпјҹеӣ дёә Mono е’Ң.NET йғҪеҸҜд»Ҙжү§иЎҢ IL д»Јз ҒпјҢжүҖд»ҘжҲ‘з”Ё Mono жқҘдҪңдёә.NET зЁӢеәҸзҡ„иҝҗиЎҢж—¶жҳҜдёҖдёӘйЎәзҗҶжҲҗз« зҡ„жғіжі•гҖӮз”ұдәҺ.NET зЁӢеәҸйңҖиҰҒ.NET Framework жҸҗдҫӣиҝҗиЎҢж”ҜжҢҒпјҢжүҖд»ҘеҪ“зӣ®ж Үи®ҫеӨҮжІЎжңүе®үиЈ….NET Framework жҲ–иҖ….NET Framework зүҲжң¬дёҚеҜ№зҡ„ж—¶еҖҷпјҢжҲ‘д»¬зҡ„зЁӢеәҸйғҪж— жі•йЎәеҲ©иҝҗиЎҢгҖӮејәиҝ«з”ЁжҲ·е®үиЈ….NET жЎҶжһ¶ж— з–‘дјҡеҪұе“Қз”ЁжҲ·дҪ“йӘҢпјҢеңЁ Windows XP е°ҡжңӘеҒңжӯўжңҚеҠЎеүҚпјҢеӣҪеҶ…иҪҜд»¶еҺӮе•ҶдёәдәҶе…је®№иҝҷдәӣз”ЁжҲ·пјҢйҖҡеёёдјҡйҖүжӢ© C++иҝҷзұ»иҜӯиЁҖжқҘзј–еҶҷеҺҹз”ҹеә”з”ЁпјҢиҝҷе°ұйҖ жҲҗдәҶеӣҪеҶ….NET жҠҖжңҜй•ҝжңҹдёҚиў«йҮҚи§Ҷзҡ„зҺ°зҠ¶гҖӮ

иҖғиҷ‘.NET зүҲжң¬зҡ„е…је®№

еңЁиҖғиҷ‘дҪҝз”Ё Mono жқҘдҪңдёә.NET еә”з”ЁзЁӢеәҸзҡ„иҝҗиЎҢж—¶еүҚпјҢйҰ–е…ҲжҲ‘д»¬жқҘиҖғиҷ‘.NET зүҲжң¬зҡ„е…је®№й—®йўҳгҖӮеҒҮи®ҫжҲ‘д»¬дҪҝз”Ё.NET Framework 3.5 зүҲжң¬з”ҹжҲҗдәҶдёҖдёӘеә”з”ЁзЁӢеәҸпјҢйӮЈд№ҲиҝҷдёӘеә”з”ЁзЁӢеәҸе°ҶеңЁе®үиЈ…дәҶ.NET Framework v3.5 зҡ„и®Ўз®—жңәдёҠиҝҗиЎҢпјҢеҰӮжһңзӣ®ж Үи®Ўз®—жңәдёҠжІЎжңүе®үиЈ….NET Framework v3.5пјҢиҝҷдёӘеә”з”ЁзЁӢеәҸе°Ҷж— жі•жӯЈеёёиҝҗиЎҢгҖӮиҝҷдёӘж—¶еҖҷпјҢжҲ‘д»¬еҸҜд»ҘжңүдёӨз§Қи§ЈеҶіж–№жЎҲпјҡз¬¬дёҖз§ҚпјҢејәиҝ«з”ЁжҲ·е®үиЈ….NET Framework v3.5пјҢж— и®әжҳҜе°ҶиҜҘжЎҶжһ¶йӣҶжҲҗеҲ°е®үиЈ…еҢ…дёӯпјҢиҝҳжҳҜеңЁе®үиЈ…иҪҜд»¶зҡ„ж—¶еҖҷиҮӘеҠЁд»ҺзҪ‘дёҠдёӢиҪҪе®үиЈ…пјҢжҳҫз„¶иҝҷз§Қж–№ејҸйғҪдјҡеҪұе“Қз”ЁжҲ·зҡ„дҪҝз”ЁдҪ“йӘҢи®©з”ЁжҲ·еҜ№еә”з”ЁзЁӢеәҸзҡ„еҚ°иұЎеӨ§жү“жҠҳжүЈпјӣз¬¬дәҢз§ҚпјҢе°қиҜ•и®©еә”з”ЁзЁӢеәҸе’Ң.NET зүҲжң¬е…је®№гҖӮжҲ‘д»¬зҹҘйҒ“ Android зЁӢеәҸжңүдёҖдёӘжңҖдҪҺ API зүҲжң¬зҡ„и®ҫзҪ®пјҢиҝҷж ·иғҪеӨҹдҝқиҜҒеә”з”ЁзЁӢеәҸеңЁдёҚдҪҺдәҺиҜҘ API зүҲжң¬зҡ„и®ҫеӨҮдёҠиҝҗиЎҢгҖӮиҝҷйҮҢжҲ‘д»¬йҖүжӢ©иҝҷз§ҚжҖқи·ҜпјҢеңЁ.NET зЁӢеәҸдёӯпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮеә”з”ЁзЁӢеәҸй…ҚзҪ®ж–Үд»¶дёӯзҡ„ supportedRuntime иҠӮзӮ№жқҘжҢҮе®ҡеә”з”ЁзЁӢеәҸиҝҗиЎҢзҡ„.NET Framework зүҲжң¬гҖӮдҫӢеҰӮдёӢйқўзҡ„й…ҚзҪ®ж–Үд»¶иғҪеӨҹдҝқиҜҒеә”з”ЁзЁӢеәҸеңЁ.NET Framework v2.0 еҲ° v3.5 й—ҙзҡ„зүҲжң¬дёҠиҝҗиЎҢгҖӮ

иҷҪз„¶иҜҙиҝҷж ·иғҪеӨҹдҝқиҜҒеә”з”ЁзЁӢеәҸзҡ„е…је®№жҖ§пјҢеҸҜжҳҜдҪ иҝҷдёӘеә”з”ЁзЁӢеәҸзҡ„е‘ҪиҝҗеҚҙжҳҜжҺҢжҸЎеңЁ.NET Framework жүӢйҮҢзҡ„пјҢеҰӮжһңз”ЁжҲ·зҡ„и®Ўз®—жңәдёҠжІЎжңүе®үиЈ….NET Framework жҲ‘д»¬дёҖж ·иҝҳжҳҜжІЎиҫҷе„ҝпјҢйӮЈд№ҲжҖҺд№ҲеҠһе‘ўпјҹжҲ‘д»¬жқҘжҗӯе»ә Mono иҝҗиЎҢж—¶гҖӮ

Mono иҝҗиЎҢж—¶зҡ„жҗӯе»ә

жҲ‘д»¬еңЁеүҚйқўиҜҙиҝҮпјҢMono дё»иҰҒз”ұдёүйғЁеҲҶз»„жҲҗпјҢеҚі C#зј–иҜ‘еҷЁ(mcs.exe)гҖҒMono иҝҗиЎҢж—¶(mono.exe)е’ҢеҹәзЎҖзұ»еә“гҖӮеӣ дёәжҲ‘д»¬иҝҷйҮҢжҳҜдёәдәҶи®©зј–еҶҷзҡ„.NET еә”з”ЁзЁӢеәҸиҝҗиЎҢеңЁ Mono иҝҗиЎҢж—¶дёӯпјҢжүҖд»ҘжҲ‘д»¬иҝҷйҮҢйңҖиҰҒзҡ„жҳҜ Mono иҝҗиЎҢж—¶(mono.exe)е’ҢеҹәзЎҖзұ»еә“гҖӮжҲ‘д»¬е»әз«ӢеҰӮдёӢзҡ„зӣ®еҪ•з»“жһ„пјҡ

дёӢйқўжқҘиҜҙиҜҙиҝҷдәӣзӣ®еҪ•еҗ„иҮӘзҡ„з»“жһ„е’ҢеҠҹиғҪпјҡ

  • bin зӣ®еҪ•пјҡж”ҫзҪ® Mono иҝҗиЎҢж—¶зҡ„зӣ®еҪ•пјҢдё»иҰҒж”ҫзҪ® mono.exeгҖҒmono-2.0.dllгҖҒlibgio-2.0-0.dllгҖҒlibglib-2.0-0.dllгҖҒlibgthread-2.0-0.dll е…ұ 5 дёӘж–Үд»¶гҖӮ
  • lib зӣ®еҪ•пјҡж”ҫзҪ® Mono дҫқиө–еә“зҡ„зӣ®еҪ•пјҢдё»иҰҒж”ҫзҪ®.NET еә“зӣ®еҪ•(жӯӨеӨ„д»Ҙ 4.0 дёәдҫӢ)гҖҒGac еә“зӣ®еҪ•гҖӮе…¶дёӯ Gac еә“зӣ®еҪ•дёӢзҡ„ AccessibilityгҖҒMono.PosixгҖҒSystemгҖҒSystem.DrawingгҖҒSystem.Windows.Forms е…ұ 5 дёӘеӯҗзӣ®еҪ•жҳҜжҲ‘д»¬ејҖеҸ‘ WindowsForm йңҖиҰҒдҪҝз”ЁеҲ°зҡ„дҫқиө–еә“гҖӮ
  • etc зӣ®еҪ•пјҡж”ҫзҪ®жҲ‘д»¬зј–еҶҷзҡ„зЁӢеәҸеҸҠе…¶зӣёе…іж–Үд»¶пјҢдё»зЁӢеәҸзҡ„ж–Үд»¶еҗҚдёә Main.exeгҖӮ

еҘҪдәҶпјҢзҺ°еңЁжҲ‘д»¬е°ұе…·еӨҮдәҶдёҖдёӘйқһеёёиҪ»йҮҸзә§зҡ„.NET зЁӢеәҸиҝҗиЎҢзҺҜеўғ(е…¶е®һж•ҙдёӘзҺҜеўғзҡ„еӨ§е°ҸеңЁ 40M е·ҰеҸі)пјҢжіЁж„Ҹд»ҘдёҠж–Үд»¶йғҪеҸҜд»ҘеңЁе®үиЈ… Mono еңЁе…¶е®үиЈ…зӣ®еҪ•еҶ…жүҫеҲ°гҖӮж №жҚ®еҚҡдё»зӣ®еүҚдәҶи§ЈеҲ°зҡ„иө„ж–ҷжқҘзңӢпјҢйҖҡиҝҮ Mono иҝҗиЎҢж—¶жқҘиҝҗиЎҢж–Үд»¶дё»иҰҒжңүе‘Ҫд»ӨиЎҢе’ҢдёҖз§Қиў«з§°дёә Mono Embedding зҡ„ж–№жЎҲгҖӮзү№еҲ«ең°пјҢз¬¬дәҢз§Қж–№жЎҲеҸҜд»ҘзӣҙжҺҘе°ҶиҝҗиЎҢж—¶еөҢе…ҘеҲ°зЁӢеәҸеҶ…пјҢжҲ‘д»¬зҶҹжӮүзҡ„ Unity3D еј•ж“Һе°ұжҳҜе°Ҷж•ҙдёӘи„ҡжң¬зҡ„иҝҗиЎҢж—¶еөҢе…ҘеҲ°дәҶ C++зЁӢеәҸдёӯпјҢдҪҶжҳҜиҝҷз§Қж–№ејҸжҜ”иҫғеӨҚжқӮпјҢжҡӮж—¶еҚҡдё»иҝҳжІЎжңүеј„жё…жҘҡе®ғзҡ„еҶ…йғЁжңәеҲ¶пјҢжүҖд»ҘжҲ‘д»¬иҝҷйҮҢйҖүжӢ©з¬¬дёҖз§Қж–№жЎҲгҖӮеҸҜжҳҜе®ғиҰҒз”Ёе‘Ҫд»ӨиЎҢе•ҠпјҢиҝ«дҪҝжҷ®йҖҡз”ЁжҲ·жқҘдҪҝз”Ёе‘Ҫд»ӨиЎҢе·Ҙе…·жҳҜд»¶з—ӣиӢҰзҡ„дәӢжғ…пјҢе°ұеғҸжҲ‘д»¬еёёеёёиў« Git жҗһеҫ—жҷ•еӨҙиҪ¬еҗ‘дёҖж ·гҖӮйӮЈд№ҲпјҢжҲ‘д»¬е°ұз”ЁзЁӢеәҸжқҘжЁЎжӢҹе‘Ҫд»ӨиЎҢеҘҪдәҶпјҒд»Җд№Ҳпјҹз”ЁзЁӢеәҸжқҘжЁЎжӢҹе‘Ҫд»ӨиЎҢпјҹиҝҷдёӘз”Ё C#жқҘеҶҷз®ҖзӣҙдёҚиғҪжӣҙз®ҖеҚ•дәҶеҘҪеҗ—пјҹиҜ·жіЁж„ҸжҲ‘д»¬иҝҷйҮҢжҳҜдёҚиғҪдҪҝз”Ё.NET Framework йҮҢзҡ„еҠҹиғҪзҡ„пјҢеӣ дёәе®ғе°ұжҳҜдёҖдёӘеј•еҜјзЁӢеәҸеҳӣпјҢеҰӮжһңеј•еҜјзЁӢеәҸйғҪйңҖиҰҒдҫқиө–.NETпјҢйӮЈжҲ‘д»¬иҝҷдёӘзЁӢеәҸжҖҺд№Ҳжҗһе•ҠгҖӮ

еҘҪеҳӣпјҢйӮЈе°ұеҶҷ C++еҺҹз”ҹеә”з”Ёеҗ§пјҢе®ғжҳҜж— йңҖд»»дҪ•дҫқиө–зҡ„гҖӮиҖҢеңЁ C++дёӯжЁЎжӢҹе‘Ҫд»ӨиЎҢдё»иҰҒжңү WinExecгҖҒShellExecute е’Ң CreateProcess дёүз§Қж–№жі•пјҢе…ідәҺиҝҷдёүз§Қж–№жі•зҡ„ејӮеҗҢеӨ§е®¶еҸҜд»ҘиҮӘиЎҢдәҶи§ЈпјҢиҝҷйҮҢжҲ‘д»¬йҖүжӢ©жңҖз®ҖеҚ•зҡ„ WinExecгҖӮд»Јз ҒеҰӮдёӢпјҡ

#include <Windows.h>

int main(int agrc,char *args[])
{

    /* жү§иЎҢе‘Ҫд»Ө */
    WinExec("bin\\mono.exe etc\\Main.exe",SW_NORMAL);
    return 0;
}

жҲ‘д»¬е°Ҷзј–иҜ‘еҘҪзҡ„зЁӢеәҸе‘ҪеҗҚдёә Launcher.exeпјҢж”ҫзҪ®жҲ‘д»¬еүҚйқўе®ҡд№үзҡ„ Mono иҝҗиЎҢж—¶зӣ®еҪ•з»“жһ„зҡ„ж №зӣ®еҪ•дёӢпјҢиҝҷдёӘж–Үд»¶е°ҶдҪңдёәеҗҜеҠЁж–Үд»¶жҡҙйңІз»ҷз”ЁжҲ·пјҢеҪ“з”ЁжҲ·зӮ№еҮ»иҝҷдёӘзЁӢеәҸеҗҺе°ұеҸҜд»Ҙжү“ејҖдё»ж–Үд»¶ Main.exeгҖӮеҘҪдәҶпјҢзҺ°еңЁжҲ‘д»¬жқҘйӘҢиҜҒдёӢжҲ‘д»¬зҡ„жғіжі•пјҡ

дҪңдёәеҜ№жҜ”пјҢжҲ‘д»¬з»ҷеҮәжӯЈеёёжғ…еҶөдёӢзЁӢеәҸзҡ„иҝҗиЎҢжҲӘеӣҫпјҡ
